# chatbot_with_langchain

A simple Streamlit chatbot built with LangChain and OpenAI.

## Setup

1. Create a `.env` file at the project root.
2. Add your OpenAI API key:

```text
OPENAI_API_KEY=sk-...
```

3. Activate your environment:

```bash
conda activate /Users/godipally.shivakumar/Desktop/Projects/langchian_1hr/langchain_1hr
```

4. Install dependencies if needed:

```bash
pip install -r requirements.txt
```

5. Run the app:

```bash
streamlit run app1.py
```

## Notes

- Keep `.env` secret; it is ignored by git.
- Use `.env.example` to share environment variable names without secrets.
